Some examples to show how to use markdown

At the beginning, I am not familiar with markdown, so I need practice to understanding it.

  1. use '====' to show headline like h1

  2. use '----' to show subline

  3. use '#--######' show h1 -- h6

  1. code block
--use ```` for write code,
public void main()
{
    console.log("test");
}

or just format the source code

public void main()
{
    console.log("test");
}

  1. The order list just use number 1-10

  2. unorder list jsut use *

  3. for strong word use this is a strong sentence

  4. for italic word use this this a italic sentence
